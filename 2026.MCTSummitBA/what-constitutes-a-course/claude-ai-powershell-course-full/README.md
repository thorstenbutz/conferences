# Hey Claude, design a course for me



In April 2026, I used the [Claude AI](https://claude.ai) prompt to create a course. You can find the [prompt](PROMPT.md) also in this current folder. The resulting files can be found in the 'Course' subfolder. Claude AI used Opus 4.7 to generate the content.

[Marp](https://marp.app/) is a technology that creates PPTX slides from plain text MD files. The slide deck was created this way.
