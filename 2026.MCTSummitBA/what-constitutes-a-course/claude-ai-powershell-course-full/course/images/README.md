The decks reference two portraits that are **not bundled** for licensing reasons.
Please drop them into this folder with exactly these filenames before rendering:

| Filename           | Subject            | Suggested source                                                                 |
|--------------------|--------------------|-----------------------------------------------------------------------------------|
| `snover.jpg`       | Jeffrey Snover     | Wikimedia Commons — `Professional Developers Conference 2009 Technical Leaders Panel 1.jpg` (cropped to portrait), or an officially-released speaker headshot (PowerShell Summit, Channel 9, Pluralsight) |
| `payette.jpg`      | Bruce Payette      | Cover portrait from *Windows PowerShell in Action* (Manning), or his public speaker photo |
| `ps-comic.jpg`     | The original Microsoft "PowerShell Hero" comic (optional) | The MSDN landing page (`mt800824`) is retired; archived copies may be on the Wayback Machine |

If those files are missing at render time, MARP will show a broken-image
placeholder — the slide will still build. Swap in your own licensed copies
and re-render.
